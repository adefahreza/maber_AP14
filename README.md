Maber menghadirkan berbagai kategori permainan matematika dengan tingkat kesulitan yang meningkat secara bertahap.

## ğŸ¯ Manfaat Game MABER

1. Meningkatkan kemampuan berhitung dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

5. Menyediakan pembelajaran adaptif dan bertahap, di mana setiap level menyesuaikan kemampuan pemain.

### 2. ğŸ§® Fitur Kategori Permainan
Game dibagi menjadi 3 kategori utama, masing-masing berisi 20 level dengan tingkat kesulitan bertahap:

- **Aritmatika Dasar**  
  Fokus pada operasi hitung sederhana seperti penjumlahan, pengurangan, perkalian, pembagian, dan bilangan berpangkat.

  Berisi teka-teki matematika berbasis gambar, di mana setiap gambar mewakili nilai tertentu yang belum diketahui.  
  Pemain harus menalar hubungan antar gambar untuk menemukan jawabannya.  

- **Baris dan Deret**  
  Mengasah kemampuan pemain dalam mengenali dan melanjutkan pola angka dengan tingkat kesulitan yang terus meningkat.
